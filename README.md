# 📊 Weighted Underwriting Simulator

A risk-based underwriting simulation engine that models how insurers, lenders, and financial institutions evaluate applications using weighted scoring, scenario analysis, and explainable decision logic.

## Features
- Weighted multi-factor risk scoring
- Scenario-based stress testing
- Explainable underwriting decisions
- Industry-aware risk profiles
- Modular architecture for future ML integration

## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS

## Run Locally
npm install
npm run dev
